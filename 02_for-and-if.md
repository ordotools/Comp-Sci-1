# For Loops and If Statements

---

## For Loops

The basic for loop in python:
```python
for x in range(5):
    print(x)   # returns 0, 1, 2, 3, 4
```

At each iteration, the value of `x` is equal to the return of the `range()` function. Its value is first `0`, then `1`, then `2`, etc.

## If Statements

If statements run code based on conditions.

```python
a = 2
if a > 1:
    print("a is bigger than 1.")
```
